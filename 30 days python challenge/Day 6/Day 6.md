## Dictionaries And Sets
    Dictionary is a Built-in Python data type that stores data in key-value pairs.
        * Keys are unique and immutable
        * values can be any python object
    Importance of dictionary-
        * Great for lookups by identifier
            person['name'],setting['theme']
        * Ideal for structured data: records, configurations, JSON-like objects
        * Efficent for counting and grouping data
        * Commonly used in algorithms needing quick access by key
    Sets is a built-in python collection data type that stores unordered, unique elements.
        * Elements must be immutable 
        * Duplicate values are automatically removed 
        * Sets supports fast membership testing and set operations          
        * Supports mathematical set operations:
            - union: a | b
            - intersection: a & b
            - difference: a - b
            - symmetric difference: a ^ b
    Importance of sets -
        * Great for removing duplicates from data
        * Efficient for membership checks
        * useful for comparing group of items
        * Perfect for tasks that need set logic, like finding common or distinct values
        * Helps simplify code when you only care about presence, not order or count
