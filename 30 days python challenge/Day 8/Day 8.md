## Introduction to **NumPY**
# **NumPY**
NumPY stands for:
`Numerical Python`
It is used for:
- Fast Calculations
- Working with arrays
- Mathematical Operations
- Data Analysis
- Machine learning

---

# Why NumPY matters
Normal Python list are slower for numerical analysis.

NumPY provides:
- Faster computation
- Less memory usage
- Vectorized operations
- Advanced mathematical functions

NumPY is the foundation for:
- Pandas
- Scikit-learn
- Tensor Flow
- Data science workflows
