## List and Tuples
 list - A list is an ordered and changeable (mutable) collection.
 Features of Lists
    --Ordered → items keep their position
    --Mutable → you can change values
    --Allows duplicate values
    --Can store different data types
Tuples - A tuple is an ordered but unchangeable (immutable) collection.
 Features of Tuples
    --Ordered
    --Immutable → cannot change values
    --Faster than lists
    --Allows duplicates
