# Day - 2
## - Arthemetic operators = symbols used to perform mathematical operations on numbers.
Some common arithmetic operators include:
- (+) (addition): adds two numbers together (e.g., 2 + 3 = 5)
- (-) (subtraction): subtracts one number from another (e.g., 5 - 2 = 3)
- (*) (multiplication): multiplies two numbers together (e.g., 2 * 3 = 6)
- (/) (division): divides one number by another (e.g., 6 / 2 = 3)
- (%) (modulus): returns the remainder of a division operation (e.g., 7 % 3 = 1)
- (**) (exponentiation): raises a number to the power of another (e.g., 2 ** 3 = 8)
- (//) (floor division): divides one number by another and rounds down to the nearest integer (e.g., 7 // 3 = 2)
## - Comparison operators = symbols used to compare two values and return a boolean result (True or False).
Some common comparison operators include:
- (==) (equal to): checks if two values are equal (e.g., 5 == 5 is True, 5 == 3 is False)
- (!=) (not equal to): checks if two values are not equal (e.g., 5 != 3 is True, 5 != 5 is False)
- (>) (greater than): checks if one value is greater than another (e.g., 5 > 3 is True, 3 > 5 is False)
- (<) (less than): checks if one value is less than another (e.g., 3 < 5 is True, 5 < 3 is False)
- (>=) (greater than or equal to): checks if one value is greater than or equal to another (e.g., 5 >= 3 is True, 3 >= 5 is False)
- (<=) (less than or equal to): checks if one value is less than or equal to another (e.g., 3 <= 5 is True, 5 <= 3 is False)
## - logical operators = symbols used to combine multiple boolean expressions and return a boolean result.
Some common logical operators include:
- and: returns True if both expressions are True (e.g., True and True is True, True and False is False)
- or: returns True if at least one expression is True (e.g., True or False is True, False or False is False)
- not: returns the opposite of a boolean expression (e.g., not True is False, not False is True)
